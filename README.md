# Tower Defense v0.1

Erster spielbarer Godot-Prototyp.

## Enthalten
- Endlose Gegner
- Bauplätze
- Automatisch schiessende Türme
- Gold, Leben, Erfahrung und Level
- Bei jedem Level-up drei zufällige Entscheidungen
- Verbesserungen gelten für alle Türme
- Elitegegner
- Game Over und Neustart

## Starten
1. Godot 4 öffnen.
2. `Import` auswählen.
3. Die Datei `project.godot` in diesem Ordner auswählen.
4. Projekt öffnen.
5. Oben rechts auf Play drücken oder F6/F5 verwenden.

## Steuerung
- Linksklick auf einen runden Bauplatz: Turm bauen
- Ein Turm kostet 100 Gold
- Bei Level-up eine der drei Karten anklicken
- Nach Game Over klicken, um neu zu starten
